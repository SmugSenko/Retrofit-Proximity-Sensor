# Retrofit-Proximity-Sensor
Freshman Design Expo Project

Purpose:
    This saftey device is a drop-in vehicle proximity sensor using mmWave sensors, controlled by a rasberryPi. The sensors output to a scalable TKInter GUI that visually identifies the distance of an object relative to the vehicle. This gui is able to accomidate as many sensors as serial lanes avalible. Also provided is arduino code for a multiplexer if you want such a device.
